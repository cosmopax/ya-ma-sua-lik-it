# Rift Relay

[![CI](https://github.com/cosmopax/ya-ma-sua-lik-it/actions/workflows/ci.yml/badge.svg)](https://github.com/cosmopax/ya-ma-sua-lik-it/actions/workflows/ci.yml)

A fast run-based action game for Reddit built with GameMaker WASM + Devvit Web.
Each run rolls mutators and challenge rules, then scores are pushed to live leaderboards with persistent profile progression.

Project slug and Devvit app identifier: `ya-ma-sua-lik-it`.

## Get Started

For instructions on how to get setup as a Reddit developer and start deploying GameMaker games to reddit, see [How To Build](docs/HowToBuild.md)

For GameMaker-to-Devvit API integration examples (save/load state, run lifecycle, perks, scores, leaderboard), see [GameMaker Bridge](docs/GameMakerBridge.md).
